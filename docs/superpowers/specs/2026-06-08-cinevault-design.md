**日期：** 2026-06-08

## 一、项目定位

黑色极简风格的全球电影社区，集电影浏览发现、个人记录、电影史与流派探索于一体。
视觉对标 A24 官网式的暗夜艺术感（深黑底色 + 金色点缀 + 大量留白）。
通过 PWA 实现移动端 App 级体验，一套代码响应式适配手机到桌面。

---

```
首页 (/)
├── 正在热映（横滑卡片）
├── 本周高分（竖排列表 Top 10）
├── 即将上映
└── 今日推荐（随机一部高分冷门）

发现 (/discover)

├── 排序：热门 / 评分最高 / 上映日期 / 片名
└── 电影网格（手机 2 列，桌面 4 列）

影史 (/history) 🆕
├── 流派入口卡片列表
│   └── 每个流派：代表海报拼图 + 名称 + 年代 + 一句话介绍
├── 点击进入流派详情页 (/history/[movement])
│   ├── 流派介绍长文（~500 字）
│   ├── 关键导演列表（头像 + 简介）
│   ├── 代表作品（可点击进入电影详情）
│   ├── 风格特点标签
│   └── 影响与传承
└── 时间线模式（可选切换 — 电影史大事年表）

电影详情 (/movie/[slug])
├── 全宽海报 + 底部渐变遮罩
├── 标题 / 导演 / 年份 / 类型 / 地区
├── 评分星级 + 评分分布条
├── 简介
├── 操作按钮组：♥喜欢 · 📌想看 · ✓看过 · ☆收藏
├── 演职员列表
├── 所属流派（可点击跳转）
└── 相关推荐

喜欢 (/likes)
├── 所有标记"喜欢"的电影网格
└── 空状态引导

我的 (/profile)
├── 看过 / 想看 / 收藏 计数卡片
├── 评分分布图
├── 对应列表入口
└── 观影统计

搜索 (/search)
├── 全屏覆盖搜索页
├── 输入联想建议下拉
└── 结果列表（含流派匹配）
```

## 四、核心功能

### 浏览发现
- 首页四个推荐维度（热映 / 本周高分 / 即将 / 今日推荐）
- 发现页多条件筛选 + 多种排序方式
- 无限滚动加载

### 搜索
- 片名 / 导演 / 演员搜索
- 输入时实时联想建议
- 🆕 可搜索流派名称，返回相关流派页

### 电影详情
- 沉浸式全宽海报 + 渐变遮罩
- 评分星级可视化 + 分布条形图
- 相关电影推荐
- 🆕 底部分享按钮 + 所属流派入口

每个流派页面包含：简介、关键导演、代表作品列表、美学风格标签、对后世的影响。

### 个人标记（localStorage 持久化）
- ♥ **喜欢** — 手动标记，金色心形，独立维度
- 📌 **想看** — 待看清单
- ✓ **看过** — 已看记录（可同时标记喜欢）
- ☆ **收藏** — 片单收藏
- ⭐ **评分** — 1-5 星

### 标记逻辑
- "看过"和"喜欢"独立——看过不一定喜欢
- 喜欢 + 高分 = 自动生成"我的最爱"片单
- 数据仅存本地，换浏览器会丢失

## 五、视觉规范

| 元素 | 值 |
|------|-----|
| 背景主色 | `#0a0a0a` |
| 卡片色 | `#141414` / `#1a1a1a` |
| 强调金 | `#c8a951` |
| 主文字 | `#ffffff` |
| 辅助文字 | `#9ca3af` |
| 圆角 | 卡片 8px / 按钮 4px |
| 间距 | 模块间距 ≥ 48px |
| 过渡 | hover 300ms ease |
| 标题字体 | Geist / Noto Serif SC（衬线） |
| 正文字体 | Inter（无衬线） |
| 设计原则 | Mobile First，触控优先，留白充裕 |

### 🆕 影史页特殊视觉
- 流派卡片使用代表电影海报拼图（2-3 部拼接）
- 时间线模式使用金色竖线 + 节点圆圈
- 流派详情页大图 Banner + 文字排版参照艺术杂志

## 八、项目范围

## 九、移动端适配

### 手机（< 640px）
- 底部固定 Tab Bar（5 项），高度 56px
- 电影网格 2 列
- 详情页海报全宽
- 搜索页全屏覆盖
- 触控热区 ≥ 44px
- 影史页竖排卡片列表

### 平板（640px - 1024px）
- 底部导航改为顶部 Tab
- 电影网格 3 列
- 影史卡片 2 列

### 桌面（> 1024px）
- 左侧固定导航栏
- 电影网格 4 列
- 详情页海报左、信息右双栏
- 影史卡片 3 列 + 侧边时间线
- 最大内容宽度 1280px 居中
